# awesome-openwrt

> 阅读: [https://awesome-openwrt.github.io/](https://awesome-openwrt.github.io/)

## 部署说明

```bash
# pre ruby and gem
ruby -v
gem -v

# install bundle and jekyll
gem install bundle
gem install jekyll

bundle -v
jekyll --version

# install dependency for this project
bundle install

# run
jekyll server
## or
bundle exec jekyll server
```
